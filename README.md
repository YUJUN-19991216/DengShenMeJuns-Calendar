# 等什么君纪念日历 🎂

一个温馨的 HTML5 日历网页，专为我的偶像 **等什么君** 制作。

每天都能看到距离 **1999年12月16日**（等什么君生日）的天数，让我们一起陪伴偶像走过的每一天。

## ✨ 功能特点

- **精确天数计算**：每一天下方清晰显示距离1999.12.16的天数（之前为负数，之后为正数）
- **生日高亮**：每年12月16日自动高亮显示，并带有 🎂 标记
- **流畅翻页**：支持上/下月翻页、上/下一年翻页，一键返回今天
- **美观设计**：粉色系温柔风格，适配手机与电脑
- **纯前端实现**：无需服务器，直接打开 HTML 文件即可使用
- **响应式布局**：在手机、平板、电脑上都能良好显示

## 🚀 使用方法

1. 下载项目或直接克隆仓库
2. 双击打开 `index.html` 文件（推荐使用 Chrome / Edge / Firefox 浏览器）
3. 即可看到当前月份的纪念日历

或者你可以直接把 `index.html` 部署到 GitHub Pages、Vercel、Netlify 等平台，生成一个专属在线链接。
